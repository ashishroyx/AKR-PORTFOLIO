Excited to launch **Respawn** – a game-inspired social web app that gives **gamers**, **creators**, and **streamers** a space to share, connect, and showcase their content. 🎮🔥

Respawn blends aesthetic design, player-first UX, and real-time data flow to simulate a truly interactive arena.

---

## 💡 What is Respawn?

Respawn is a **gaming content platform** where users can post updates, highlight games, and create a digital identity around their gaming interests. Whether you're a content creator or a casual player, Respawn makes expression seamless.

### 🧩 Core Capabilities:
- 🎮 Share and edit game-related posts with media support
- 👤 Create a gamer profile with avatar, bio, and game preferences
- 📅 Browse and interact with posts across a clean, modern feed
- ✏️ Edit and delete posts with real-time UI updates

---

## 🔧 Tech Stack & Features

- ✅ **React.js** – modern frontend with reusable components
- ✅ **Tailwind CSS** – fast UI styling with custom themes
- ✅ **Appwrite** – secure and scalable backend-as-a-service (auth, database, storage)
- ✅ **Framer Motion** – rich animations and interactive transitions
- ✅ **Lucide Icons** – clean, minimal icons for UI clarity
- ✅ **Protected Routes** – only authenticated users can post or edit
- ✅ **Dynamic Feed** – fetches and renders real-time posts

---

## 🎨 Design & UX Principles

Respawn is crafted to **look and feel like a gaming dashboard** — energetic, responsive, and fun to use. Animations, layout, and iconography all reflect the gaming world while keeping things lightweight and fast.

- 🧱 Card-based post layout with hover highlights
- 📱 Fully responsive – optimized for mobile gamers too
- ✨ Smooth transitions using **Framer Motion**
- 🕹️ UI styled with dark/light gamer themes (customizable)
- 🧭 Clear call-to-actions and accessible design for all users

---

## 🚀 Development Highlights

- 🧠 Built using **Appwrite’s full-stack services** for auth, DB, and media uploads
- 🔐 Implemented login guards with protected route checks
- 🛠️ Modular components: **PostCard**, **UserProfile**, **EditModal**, etc.
- 🔄 Real-time updates with Appwrite subscriptions and event listeners
- 🔍 Filter logic built for future sorting (by genre, platform, etc.)
- 📤 Image upload and post formatting built with Appwrite’s storage SDK
